## Tasks Completed

### Task 1: Count Total Number of Students

Use COUNT() function to calculate the total number of students.

### Task 2: Find Average Marks of Students

Use AVG() function to calculate the average marks.

### Task 3: Find Highest and Lowest Marks

Use MAX() and MIN() functions to find highest and lowest marks.

### Task 4: Find Department-wise Average Marks

Use GROUP BY clause with AVG() function.

### Task 5: Display Departments Where Average Marks > 70

Use HAVING clause to filter grouped results.
